### [CortexGrid](https://github.com/Jonjan95/CortexGrid)

A local-first prototype exploring how developers can decide what work belongs to them and what can be delegated to Codex. It turns a development task into mode-specific execution contracts, editable prompts, visual workflows, and deterministic simulations.

I built CortexGrid as a short AI-assisted summer learning project after realizing that effective prompting is less about finding magic words and more about clearly defining responsibility, boundaries, and validation.
